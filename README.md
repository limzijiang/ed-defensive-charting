# 急診防衛性病歷寫作 Orientation

給急診未來 V1 的上工前 orientation：防衛性病歷寫作範本集。

🌐 **線上版：https://limzijiang.github.io/ed-defensive-charting/**

內容：
- 出院病摘 discharge notes（含 GRACE-3、AHA/ASA guideline 引用）
- SDM / AAD / 轉院記錄寫法
- 檢查與處置的告知（顯影劑、侵入性處置、病危告知）
- 各主訴 HPI / PE 模板（外傷、內科、神經、兒科）
- AutoHotkey v2 快速鍵腳本（`ed-note-templates.ahk`）

⚠️ 範本僅供參考，務必依病人實際狀況修改。不含任何真實病人資料。
